与えられたドキュメントに従って、既存の仕様を修正してください。

1. ドキュメントは docs ディレクトリにあります。引数で指定されたドキュメントから、対応する revision 番号のセクションを探してください。たとえば revision number が 2 だった場合は、 # revision 2 というセクションを探して読んでください。
2. 指示された仕様変更を実装してください。
3. 必要があると判断できる場合は、サンプルプログラムの実装を修正してください。
4. dll 本体、および、サンプルプログラムがビルドできることを確認してください。ビルドできなくなっていたら修正してください。
