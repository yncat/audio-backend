与えられたドキュメントに従って、新しい機能を実装します。
1. ドキュメントは docs ディレクトリにあります。引数で指定されたドキュメントを読み、内容を理解してください。
2. 特定の見出しのタイトルが引数に指定されていれば、そのセクションおよびそれに依存する範囲のみを実装対象としてください。ドキュメントだけが与えられている場合は、全体を実装してください。
3. テスト用プログラムに、追加した機能をテストするための画面を追加してください。ただし、これは、dllの機能が追加/変更された場合に限ります。プロジェクト自体の構成変更や、サンプルプログラム自身の仕様変更については、サンプルを変更する必要がないので、スキップしてください。追加画面を実装するときは、今回実装した機能を実際に触って試せるようにしてください。
